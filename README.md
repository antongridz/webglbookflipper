# WebGL Book Flipper

Интерактивный 3D флиппер книг с использованием Three.js, React и Vite.

## Описание

Проект представляет собой интерактивное портфолио с 3D книгой, страницы которой перелистываются при скролле. Книга реагирует на движение мыши, создавая реалистичный эффект наклона.

## Технологии

- **React** - UI библиотека
- **Three.js** - 3D графика и анимация
- **Vite** - Сборщик и dev-сервер
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **shadcn/ui** - UI компоненты

## Установка

```bash
# Установка зависимостей
pnpm install
```

## Запуск

```bash
# Запуск dev-сервера
pnpm dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
# Сборка для продакшена
pnpm build

# Предпросмотр продакшен сборки
pnpm preview
```

## Особенности

- Плавная анимация перелистывания страниц при скролле
- Интерактивный наклон книги при движении мыши
- Динамическая загрузка Three.js для оптимизации производительности
- Адаптивный дизайн
